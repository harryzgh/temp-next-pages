# 版本相关 [Next.js](https://nextjscn.org/docs/pages/getting-started/installation)

1. 使用 `create-next-app`框架安装
2. next 版本 15.3.0

## 项目启动

项目安装好后，使用以下命令进行启动

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
